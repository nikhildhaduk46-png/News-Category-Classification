# News Category Classification using NLP

## Project Overview

This project is a Natural Language Processing (NLP) application that automatically classifies news headlines into different categories such as Politics, Sports, Business, Entertainment, Technology, and more.

The goal is to build a machine learning model capable of understanding textual data and predicting the correct news category based on the headline content.

---

## Dataset

The dataset contains cleaned news headlines and their corresponding categories.

### Features

* News Headline
* News Category

### Target Variable

* Category

---

## Technologies Used

* Python
* NumPy

---

## NLP Preprocessing Steps

1. Convert text to lowercase
2. Remove punctuation
3. Remove numbers
4. Remove emojis
5. Tokenization using NLTK
6. Stopword removal
7. TF-IDF Vectorization

---

## Machine Learning Models

The following algorithms were used and compared:

* Logistic Regression
* Multinomial Naive Bayes
* Random Forest Classifier
